# ACC Architecture & Interior - v2 (Archix Edition)

Esta es una nueva versión del sitio web de **Africana de Contratas y Construcciones (ACC)**, rediseñada con una estética moderna inspirada en la plantilla *Archix*.

## 🎨 Características de Diseño
- **Estilo**: Arquitectura & Interiorismo moderno, minimalista y elegante.
- **Paleta de Colores**: 
    - Fondo: Blanco (#FFFFFF)
    - Accento: Ocre/Beige (#C5A880)
    - Textos: Negro (#000000) y Gris (#5E5E5E)
- **Tipografías**: 
    - **Poppins**: Títulos y encabezados.
    - **Roboto**: Cuerpo de texto y párrafos.

## 🚀 Tecnologías
- **Core**: HTML5 Semántico.
- **Framework**: Bootstrap 5.
- **Animaciones**: AOS (Animate on Scroll) para transiciones fluidas.
- **Iconos**: Font Awesome 6.
- **Build Tool**: Vite (Vanilla JS).

## 🛠️ Instalación y Desarrollo
1. Entrar en la carpeta: `cd acc-archix`
2. Instalar dependencias: `npm install`
3. Iniciar servidor de desarrollo: `npm run dev`
4. Construir para producción: `npm run build`

## 📂 Estructura
- `index.html`: Estructura principal y secciones.
- `src/style.css`: Estilos personalizados y variables de diseño.
- `src/main.js`: Lógica de animaciones y filtrado.
